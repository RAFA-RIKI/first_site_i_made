About RAFA-RIKI Project

This application is a dynamic web project developed using Python and the Flask framework. It serves as a personal learning dashboard to practice database integration and user management.

Technology Stack

The application relies on a modern stack for development:

  • Backend: Python & Flask
  • Database: SQLAlchemy (SQLite)
  • Frontend: Jinja Templating & Tailwind CSS

Key Features

This project implements essential web application features:

  • User Registration & Login (Secure Hashing)
  • CRUD Operations (Create, Read, Delete)
  • User Authentication and Authorization
  • Real-time Data Dashboard
